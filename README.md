# Metal Renderer (maybe minecraft 2)

## Running locally

```bash
git clone https://github.com/tonadr1022/metalrender
cd metalrender
git submodule update --init --recursive
cmake --preset Release
cmake --build build/Release
./bin/Release/metalrender
```

- Use CMake presets
