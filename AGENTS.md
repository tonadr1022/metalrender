# Agent Instructions

## Build and Run

To build and run the project, use the following commands.

### Configure

Use the `Debug` preset from `CMakePresets.json`:

```bash
cmake --preset Debug
```

### Build

Build the `Debug` configuration:

```bash
cmake --build build/Debug --target metalrender
```

### Run

Execute the binary:

```bash
./build/Debug/bin/metalrender
```
